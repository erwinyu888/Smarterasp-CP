# Rebuild Blockers

Record features that cannot be completed because an exact referenced Classic ASP dependency is missing, a live gateway is unavailable, or a real-data test cannot be completed safely.

## Working Rule

- Use the exact file referenced by the Classic ASP feature being rebuilt.
- Ignore code inside `if 1 = 2 then ... end if` blocks because the latest ASP uses that pattern for intentionally disabled legacy behavior.
- If that referenced file is missing, stop that feature and log it here.
- Do not use similar files from other folders or older copies unless the exact latest source chain references them.
- Continue with the next independent feature after logging the blocker.
